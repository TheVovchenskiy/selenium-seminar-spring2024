# selenium-seminar-spring2024

## Credentials

Для использования данных для входа в личный кабинет, в файл `./Selenium/code/files/userdata` добавь их в следующем формате:

```json
{
    "user": "your_login",
    "password": "your_password"
}
```

## Запуск

Для запуска тестов, используй:

```bash
make test-park
```
